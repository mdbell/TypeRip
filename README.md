# TypeRip
Web-based Adobe TypeKit font ripper
##[Get It Here](http://codezombie.github.io/TypeRip/)

###How to use it
  1. Enter a TypeKit font family url and press enter or click 'rip fonts'
  2. Click on the font you want to download
  3. That's it.
